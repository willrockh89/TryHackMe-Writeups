# Defensive Security Intro

## Overview
This room introduces defensive security and how organizations protect systems against attacks in real-world environments.

## Key Concepts
- Blue team responsibilities
- Monitoring and detection
- Incident response
- Security tooling

## Learning Objectives
- Understand defensive security roles
- Learn how attacks are detected
- Identify common defensive technologies
- Understand incident response basics

## High-Level Steps
1. Overview of defensive security and blue team roles
2. Explanation of monitoring and alerting
3. Introduction to detection technologies
4. Discussion of real-world defense scenarios

## Technologies Covered
- SIEM platforms
- Firewalls
- Intrusion detection systems (IDS)
- Logging and monitoring tools

## Takeaways
- Defensive security is proactive and reactive
- Visibility and logging are critical
- Detection speed matters as much as prevention

## Notes
This room complements Offensive Security Intro by showing the defensive side of the security lifecycle.
